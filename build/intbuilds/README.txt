********************************************************************************************************

This location contains the BATCH files that are written to do the following; 

(1) Build the Maven Module Projects 
(2) Copy the following into the "qaauto-runnable-jars" folder
	(a) built Projects 
	(b) Dependent jars
	(c) testNG XML Suites 
	(d) supporting resources such as databanks, object repositories, XML templates and 
	    environment configuration 

********************************************************************************************************

********************************************************************************************************
Please update your repository root path in config.properties file before running any of 
the batch scripts. Root Path is the location in which the code is checked out. 

Example: If the automation repository in your system is D:\Santhoshi\Automation\ws\........ then 
the root path in config.properties file should be D:\Santhoshi\Automation\
